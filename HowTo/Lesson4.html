Lesson 4: Routing

Ember.js prides itself on the use of urls to be able to share the experience of using your website with others. Routing adds text to the end of the url so that the user can simply send the url to another user and the second user will see what the first user sees. Below is some example text of how to set a route:

figure 1 taken from http://guides.emberjs.com/v1.10.0/routing/defining-your-routes/

This code will add two different url routes to the end of the websites url. Typing yourwebsite.com/about will bring open the about page in your web site and yourwebsite.com/favs will bring the favs page up on your website.
