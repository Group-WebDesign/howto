Class

How to create a new class in EmberJS? Simple, just use Ember.Object.extend() function to create a new ember class! The picture below shows how to define a new class named Person with a method of say() in it.


Now, what if user wants to create a subclass or wants to define a class after creating one? Take a look at the snippet of code below carefully and follow the explanation.


First, the codeline 1 to 6 basically similar to the previous example which just defining a new class name person. Then in codeline 8 to 12, the user is creating a subclass. However, there is one requirement if want to use subclass, user must call special _super() method in order to access the implementation of parent class. Next, in codeline 14 to 16, the user is creating an instance from that "Soldier" subclass by calling a create() method. This simply means that all the properties from the predefined class will be available to the instance that user just created. That's why in the last codeline 18, Yehuda Katz will say "Yes" (from the paramater)and "Sir" from the predefined code in subclass.
