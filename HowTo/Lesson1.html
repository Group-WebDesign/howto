Lesson 1: Class

How to create a new ember class?

Use Ember.Object.extend() function to create a new ember class!

Just like othe OOP language, you can define a child in ember!

What's more, you can even use a special _super() method to call the parent function that has been overided in the child.

Code credit: Ember.js official guide, "http://guides.emberjs.com/v1.13.0/object-model/classes-and-instances/"
